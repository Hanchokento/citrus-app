# kaisha_memo_nakazawa

## 会社設立までのステップ

1. 会社設立に必要な基礎情報を決定する
   * 会社形態
        株式会社,合同会社,合名会社,合資会社
   * 商号（会社名）
   * 事業目的
   * 本店所在地
   * 資本金
        下限はなし
   * 会社設立日
   * 会計年度
   * 役員や株主の構成
2. 会社用の印鑑（実印）を作成する
   5000-50000くらい
3. 定款を作成する
   会社のルールブック
   絶対必要事項
    事業の目的，商号，本社所在地，資本金額（出資財産額），発起人の氏名と住所
4. 公証役場で定款の認証を受ける
   認証手数料：1万5,000〜5万円（※）
   謄本代：250円×定款の枚数（現金で持参）
   収入印紙：4万円（電子定款は不要）
5. 資本金の払い込みを行う
   発起人の個人口座が最初
6. 登記申請書類を用意し登記申請する
7. 設立後
   税務署への届出
   社会保険・労働保険の手続き
   法人用の口座開設手続き
